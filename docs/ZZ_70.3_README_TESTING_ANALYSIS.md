## 🚀 COMMANDS THƯỜNG DÙNG

```bash
# ============================================
# CHẠY TESTS
# ============================================

# Unit tests only
pnpm test:unit

# Integration tests
pnpm test:integration

# E2E tests
pnpm test:e2e

# Test một module cụ thể
pnpm test src/routes/wishlist/__tests__

# Test một file cụ thể
pnpm test src/routes/wishlist/__tests__/wishlist.service.spec.ts

# ============================================
# COVERAGE
# ============================================

# Chạy tests với coverage
pnpm test:cov

# Xem coverage report trong browser
open coverage/lcov-report/index.html

# ============================================
# DEVELOPMENT
# ============================================

# Watch mode (auto re-run khi file thay đổi)
pnpm test:watch

# Debug tests
pnpm test:debug

# CI/CD tests
pnpm test:ci
```
